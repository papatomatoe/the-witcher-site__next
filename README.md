## Develop

Clone project with:

```bash
git clone <REPOSITORY>
```

Installed dependencies with

```bush
npm install
```

Start a development server:

```bash
npm run dev
```

## Building

To create a production version of your app:

```bash
npm run build
```

You can preview the production build with

```bush
npm run start
```
